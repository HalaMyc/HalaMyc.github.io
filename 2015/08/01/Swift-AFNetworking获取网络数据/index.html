Swift+AFNetworking获取网络数据

Aug 1, 2015

苹果14年推出Swift这门语言时吸引了众多开发者的目光，15年wwdc大会上将Swift开源后，这门语言注定会在几年后成为一门伟大的语言，也会成为iOS开发的主流语言
AFNetworking是iOS开发中最常用的第三方库之一，有关AFNetworking可以访问AFNetworking的github
通过这篇文章你可以学习到如何使用Swift+AFNetworking获取网络数据

第一步，使用CocoaPods导入AFNetworking

CocoaPods是一个Swift和Objective-c的cocoa项目的第三方库的管理工具，iOS和OS X程序员都很有必要掌握这个工具的使用方法，具体使用方法可以参考CocoaPods安装和使用教程,文章中就使用了AFNetworking，可见这个库的重要程度。

第二步，新建Swift与OC混编所需要的桥接文件


创建一个.h文件，命名随便，我这里命名为Bridging-header.h
接下来需要在bulid settings中设置桥接文件

路径为项目名称/文件名
如果觉得麻烦的话，在项目中创建一个OC文件，xcode为自动帮你生成一个桥接文件并设置好路径
最后，打开桥接文件，在文件中导入AFNetworking就行了

#import <AFNetworking/AFNetworking.h>

第三步，在代码中使用AFNetworking
let manager = AFHTTPRequestOperationManager()
 manager.GET("your api", parameters: [], success: { (operation:AFHTTPRequestOperation!, response: AnyObject!) -> Void in
        }) { (operation: AFHTTPRequestOperation!, error: NSError!) -> Void in
   }
